Pyhealth.data 结构

  • [README]: This module defines the basic data structures in PyHealth.
  • [Structures]
    • Event is the info structure for a single clinical event
    • Visit is the info structure for a single clinical visit
    • Patient is the info structure for a single patient

Event

  • 【变量】：

  • code`：事件的代码，为字符串类型（例如，心脏衰竭的代码可以是 “428.0”）。

  • table：记录事件的表的名称，为字符串类型（例如，”DIAGNOSES_ICD”）。

  • vocabulary：代码的词汇表，为字符串类型（例如，’ICD9CM’、’ICD10CM’、’NDC’）。

  • visit_id：访问的唯一标识符，为字符串类型。

  • patient_id：患者的唯一标识符，为字符串类型。

  • timestamp（可选）：事件的时间戳，为可选的 datetime 类型。默认为 None。

  • **attr（可选）：事件的可选属性，作为键值对添加到访问中。

  • 【属性】：显示附加属性

    attr_dict：显示存储在 **attr 中的信息
