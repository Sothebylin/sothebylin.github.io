模板
# 

# 自顶向下递归的动态规划
def dp(状态1, 状态2, ...):
    for 选择 in 所有可能的选择:
        #此时的状态已经因为做了选择而改变
        result = 求最值(result, dp(状态1, 状态2, ...))
    return result

# 自底向上迭代的动态规划
# 初始化 base case
dp[0][0][...] = base case
# 进行状态转移
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)

  2. 确定递推公式

  3. dp数组如何初始化

P1095 [NOIP 2007 普及组] 守望者的逃离
#

题目描述
#

恶魔猎手尤迪安野心勃勃，他背叛了暗夜精灵，率领深藏在海底的娜迦族企图叛变。

守望者在与尤迪安的交锋中遭遇了围杀，被困在一个荒芜的大岛上。

为了杀死守望者，尤迪安开始对这个荒岛施咒，这座岛很快就会沉下去。到那时，岛上的所有人都会遇难。

守望者的跑步速度为 $17m/s$，以这样的速度是无法逃离荒岛的。庆幸的是守望者拥有闪烁法术，可在 $1s$ 内移动 $60m$，不过每次使用闪烁法术都会消耗魔法值 $10$ 点。守望者的魔法值恢复的速度为 $4$ 点每秒，只有处在原地休息状态时才能恢复。

现在已知守望者的魔法初值 $M$，他所在的初始位置与岛的出口之间的距离 $S$，岛沉没的时间 $T$。你的任务是写一个程序帮助守望者计算如何在最短的时间内逃离荒岛，若不能逃出，则输出守望者在剩下的时间内能走的最远距离。

注意：守望者跑步、闪烁或休息活动均以秒为单位，且每次活动的持续时间为整数秒。距离的单位为米。

输入格式
#

输入数据共一行三个非负整数，分别表示 $M$，$S$，$T$。

输出格式
#

输出数据共两行。

第一行一个字符串 $\texttt{Yes}$ 或 $\texttt{No}$，即守望者是否能逃离荒岛。

第二行包含一个整数。第一行为 $\texttt{Yes}$ 时表示守望者逃离荒岛的最短时间；第一行为 $\texttt{No}$ 时表示守望者能走的最远距离。

输入输出样例
#

输入

39 200 4

输出

No 197

输入

36 255 10

输出

Yes 6

说明/提示

对于 $30%$ 的数据，$1 \le T \le 10$，$ 1 \le S \le 100$；

对于 $50%$ 的数据，$1 \le T \le 10^3$，$ 1 \le S \le 10^4$；

对于 $100%$ 的数据，$1 \le T \le 3\times 10^5$，$0 \le M \le 10^3$，$ 1 \le S \le 10^8$。

AC代码
# 

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int m, s, t;
    cin >> m >> s >> t;

    vector<int> f(t + 1); // 仅使用魔法的最远距离
    int magic = m;
    f[0] = 0;
    for (int i = 1; i <= t; ++i) {
        if (magic >= 10) {
            f[i] = f[i-1] + 60;
            magic -= 10;
        } else {
            f[i] = f[i-1];
            magic += 4;
        }
    }

    int max_dist = 0;
    for (int i = 1; i <= t; ++i) {
        max_dist = max(max_dist + 17, f[i]); // 两种策略取优
        if (max_dist >= s) {
            cout << "Yes\n" << i;
            return 0;
        }
    }

    cout << "No\n" << max_dist;
    return 0;
}

P3842 [TJOI2007] 线段
#

题目描述
#

在一个 $n \times n$ 的平面上，在每一行中有一条线段，第 $i$ 行的线段的左端点是$(i, L_{i})$，右端点是$(i, R_{i})$。

你从 $(1,1)$ 点出发，要求沿途走过所有的线段，最终到达 $(n,n)$ 点，且所走的路程长度要尽量短。

更具体一些说，你在任何时候只能选择向下走一步（行数增加 $1$）、向左走一步（列数减少 $1$）或是向右走一步（列数增加 $1$）。当然，由于你不能向上行走，因此在从任何一行向下走到另一行的时候，你必须保证已经走完本行的那条线段。

输入格式
#

第一行有一个整数 $n$。

以下 $n$ 行，在第 $i$ 行（总第 $(i+1)$ 行）的两个整数表示 $L_i$ 和 $R_i$。

输出格式
#

仅包含一个整数，你选择的最短路程的长度。

输入输出样例
#

输入

6
2 6
3 4
1 3
1 2
3 6
4 5

输出

24

说明/提示

我们选择的路线是

 (1, 1) (1, 6)
 (2, 6) (2, 3)
 (3, 3) (3, 1)
 (4, 1) (4, 2)
 (5, 2) (5, 6)
 (6, 6) (6, 4) (6, 6)

不难计算得到，路程的总长度是 $24$。

对于 $100%$ 的数据中，$n \le 2 \times 10^4$，$1 \le L_i \le R_i \le n$。

AC代码
# 

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    // 存储每行线段的左右端点
    vector<int> L(n+1), R(n+1);
    for(int i=1; i<=n; ++i) 
        cin >> L[i] >> R[i];

    // 特殊情况处理：只有一行
    if(n == 1) {
        int walk = R[1] - L[1]; // 必须走完线段
        int start = min(abs(1-L[1]), abs(1-R[1])); // 起点到线段的最短距离
        cout << walk + start;
        return 0;
    }

    /**************************************
    * 核心DP逻辑：
    * pl -> 上一行结束时在左端点的最短路径
    * pr -> 上一行结束时在右端点的最短路径
    ?**************************************/
    long long pl = abs(1 - R[1]) + (R[1]-L[1]); // 第一行结束在左端
    long long pr = abs(1 - L[1]) + (R[1]-L[1]); // 第一行结束在右端

    for(int i=2; i<=n; ++i) {
        int ll = L[i], rr = R[i]; // 当前行端点
        int seg = rr - ll;        // 必须走的线段长度

        // 计算转移到当前行左端的两条路径
        long long to_left = min(
            pl + abs(rr - L[i-1]) + 1 + seg, // 从上一行左端出发
            pr + abs(rr - R[i-1]) + 1 + seg  // 从上一行右端出发
        );

        // 计算转移到当前行右端的两条路径
        long long to_right = min(
            pl + abs(ll - L[i-1]) + 1 + seg, // 从上一行左端出发
            pr + abs(ll - R[i-1]) + 1 + seg  // 从上一行右端出发
        );

        pl = to_left;   // 更新状态
        pr = to_right;
    }

    // 最后要走到(n,n)
    cout << min(pl + abs(n-L[n]), pr + abs(n-R[n]));
}

leetcode63.不同路径 II
#

注意此题只能向下或向右]

题目描述
#

给定一个 m x n 的整数数组 grid。一个机器人初始位于 左上角（即 grid[0][0]）。机器人尝试移动到 右下角（即 grid[m - 1][n - 1]）。机器人每次只能向下或者向右移动一步。

网格中的障碍物和空位置分别用 1 和 0 来表示。机器人的移动路径中不能包含 任何 有障碍物的方格。

返回机器人能够到达右下角的不同路径数量。

测试用例保证答案小于等于 2 * 109。

AC代码
# 

class Solution {
public:
    int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
        int m = obstacleGrid.size();
        int n = obstacleGrid[0].size();
        if (obstacleGrid[m - 1][n - 1] == 1 || obstacleGrid[0][0] == 1) //如果在起点或终点出现了障碍，直接返回0
            return 0;
        vector<vector<int>> dp(m, vector<int>(n, 0));
        
        //初始化
        for (int i = 0; i < m && obstacleGrid[i][0] == 0; i++) dp[i][0] = 1;
        for (int j = 0; j < n && obstacleGrid[0][j] == 0; j++) dp[0][j] = 1;
        
        //开始遍历
        for (int i = 1; i < m; i++) {//i,j表示所处的是几行几列
            for (int j = 1; j < n; j++) {
                if (obstacleGrid[i][j] == 1) continue;
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }
        return dp[m - 1][n - 1];
    }
};

leetcode 343. 整数拆分
#

问题描述
#

给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

示例 1:

  • 输入: 2
  • 输出: 1
  • 解释: 2 = 1 + 1, 1 × 1 = 1。

AC代码
# 

class Solution {
public:
    int integerBreak(int n) {
        vector<int> dp(n + 1);
        dp[2] = 1;
        for (int i = 3; i <= n ; i++) {
            for (int j = 1; j <= i / 2; j++) {
                dp[i] = max(
                dp[i], max((i - j) * j, dp[i - j] * j)
                   //(i-j)*j 是划分成两个数相乘，dp[i-j]*j是划分成很多数相乘
                );
            }
        }
        return dp[n];
    }
};

96.不同的二叉搜索树
#

注意是"结构"不同

问题描述
#

力扣题目链接

给定一个整数 n，求以 1 … n 为节点组成的二叉搜索树有多少种？

示例:

class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n + 1);
        dp[0] = 1;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                dp[i] += dp[j - 1] * dp[i - j];
            }
        }
        return dp[n];
    }
};

dp[3]，就是 元素1为头结点搜索树的数量 + 元素2为头结点搜索树的数量 + 元素3为头结点搜索树的数量

元素1为头结点搜索树的数量 = 右子树有2个元素的搜索树数量 * 左子树有0个元素的搜索树数量

元素2为头结点搜索树的数量 = 右子树有1个元素的搜索树数量 * 左子树有1个元素的搜索树数量

元素3为头结点搜索树的数量 = 右子树有0个元素的搜索树数量 * 左子树有2个元素的搜索树数量

【蓝桥杯】可变半径的连续切线圆盘
#

问题描述
#

有 NN 个圆盘需要放置在一条直线上的指定位置处，每个圆盘的中心位于已知的坐标点上。每两个相邻的圆盘必须相切。对于每个圆盘，都有一个允许的最小半径和最大半径的限制。请计算出所有满足条件的圆盘排列方案的数量。

输入格式
#

第一行包含一个整数 NN，表示圆盘的数量。

第二行包含 NN 个递增的整数，代表圆盘中心的坐标。

接下来的 NN 行，每一行包含两个整数，分别表示一个圆盘的最小半径和最大半径。

输出格式
#

输出一个整数，表示满足条件的圆盘排列方案的数量。

样例输入
# 

4
0 4 7 10
1 5
1 5
1 3
1 3

样例输出
# 

2

评测数据规模
#

  • 2≤N≤1052≤N≤105
  • 圆盘中心的坐标是介于 00 和 109109 之间的整数
  • 圆盘的半径限制是介于 11 和 109109 之间的整数

笨笨枚举＋DFS+DP
# 

#include <iostream>
#include <vector>
#include <unordered_map>
using namespace std;

const int MAXN = 1e5 + 5;
const long long MOD = 1e9 + 7;

int N;
vector<int> centers;
vector<int> minR;
vector<int> maxR;
unordered_map<int, unordered_map<int, long long>> dp;

long long dfs(int i, int r) {
    if (i == N - 1) return 1; // 最后一个圆盘，返回1
    if (dp[i].count(r)) return dp[i][r]; // 如果i圆盘的r半径已经计算过方案数，直接返回

    long long res = 0;
    for (int nextR = minR[i + 1]; nextR <= maxR[i + 1]; ++nextR) {
        // 检查是否满足相切条件
        if (centers[i + 1] - centers[i] == r + nextR) {
            res = (res + dfs(i + 1, nextR)) % MOD;
        }
    }
    dp[i][r] = res; // 记忆化存储结果
    return res;
}

int main() {
    cin >> N;
    centers.resize(N);
    minR.resize(N);
    maxR.resize(N);

    for (int i = 0; i < N; ++i)cin >> centers[i];
    

    for (int i = 0; i < N; ++i) cin >> minR[i] >> maxR[i];

    long long ans = 0;
    for (int r = minR[0]; r <= maxR[0]; ++r) {
        ans = (ans + dfs(0, r)) % MOD;
    }

    cout << ans << endl;
    return 0;
}

巧用数学推导
# 

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    
    int N;
    cin >> N;
    vector<int> centers(N);
    for (int i = 0; i < N; ++i) {
        cin >> centers[i];
    }
    
    vector<int> minR(N), maxR(N);
    for (int i = 0; i < N; ++i) {
        cin >> minR[i] >> maxR[i];
    }
    
    if (N == 0) {
        cout << 0 << endl;
        return 0;
    }
    
    long long prev_low = minR[0];
    long long prev_high = maxR[0];
    if (prev_low > prev_high) {
        cout << 0 << endl;
        return 0;
    }
    long long count = prev_high - prev_low + 1;
    
    for (int i = 1; i < N; ++i) {
        int d = centers[i] - centers[i - 1];
        
        long long new_prev_low = max(prev_low, static_cast<long long>(d) - maxR[i]);
        long long new_prev_high = min(prev_high, static_cast<long long>(d) - minR[i]);
        
        if (new_prev_low > new_prev_high) {
            cout << 0 << endl;
            return 0;
        }
        
        count = new_prev_high - new_prev_low + 1;
        
        long long curr_low = d - new_prev_high;
        long long curr_high = d - new_prev_low;
        
        prev_low = curr_low;
        prev_high = curr_high;
    }
    
    cout << count << endl;
    
    return 0;
}
